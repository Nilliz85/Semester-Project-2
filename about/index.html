About

Welcome to my social media platform! I created this platform as a part of my studies in front-end development at Noroff School of Technology and Digital Media. The aim of this project was to apply my knowledge of JavaScript techniques to implement the front-end functionality for a social media application using an API provided by Noroff.

The platform allows users with @noroff.no or @stud.noroff.no email addresses to register and create their profile, and then log in to access their profile. Users can view a feed of posts, filter and search through them, view posts by ID, and create, update, or delete their own posts. In addition, there are optional features such as creating comments on posts, editing profile media, following or unfollowing other users' profiles, and reacting to posts.

I wanted to create an easy-to-use and attractive platform that meets the requirements of the project, while also implementing additional features to showcase my skills and creativity. I hope you enjoy exploring and using this platform as much as I enjoyed creating it!
